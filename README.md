# 🎨 Can Ersoy — Portfolio Website

This is my personal portfolio website showcasing selected works in **graphic design, branding, and art direction**.

The website is designed as a minimal and immersive experience, focusing on **visual impact, typography, and interaction**.

## 🧠 Concept

This portfolio explores a **dynamic visual system** where:

* Projects are presented as **full-bleed visual blocks**
* Hover interactions create a **cinematic zoom effect**
* Layout remains minimal to emphasize **art direction and imagery**

Inspired by contemporary portfolio experiences and creative studio websites.
